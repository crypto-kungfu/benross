# Meta-Cognition — Consistency Checks

Run before and after meaningful changes.

## Memory Coherence
- Does `activeContext.md` describe the current focus?
- Does `progress.md` reflect completed and pending work?
- Did a durable project rule change that belongs in `patterns.md`?
- Did a stack or deployment fact change that belongs in `techContext.md`?

## Brief Consistency
- Local SEO and conversion remain the priority.
- UK English is used.
- Wasp nest removal remains the lead service.
- Only confirmed services are published.
- The residential street address is absent.

## Scope Creep Check
- Are we adding CMS, React islands, booking systems, or analytics before they are needed?
- Are we adding pages outside the approved structure?
- Are we writing long educational copy when a concise trade-service answer is better?

## Quality Bar
- Build attempted.
- Linter diagnostics checked for edited files where available.
- No secrets or `.env` committed.
- Memory updated after meaningful work.
